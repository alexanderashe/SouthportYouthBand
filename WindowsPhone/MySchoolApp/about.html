About

This application was created by Mark Frydenberg at Bentley University to demonstrate features of Windows Phone 7 programming using Visual Studio 2010 and Visual Basic.net. It was first demonstrated at the Microsoft US Public Sector CIO Summit in February, 2011.

The application is intended to be a template for educators at other colleges and universities to use as an example in their classrooms. By replacing the Bentley-specific information (logo, colors, URLs, sports team name, and content information) in three XML files with your own, you can create a working app for your school. Those who wish can also delve into the code to make additional modifications to button labels, page titles, etc. This is an effective way to introduce Phone programming concepts to beginning students in application development courses.
